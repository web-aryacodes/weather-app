# 🌦 Weather App

A clean and responsive weather application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Search weather by city
- Real-time data using OpenWeatherMap API
- Weather icons and temperature display
- Error handling for invalid cities
- Responsive UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## 📌 Future Improvements
- 5-day forecast
- Auto location detection
- UI enhancements
